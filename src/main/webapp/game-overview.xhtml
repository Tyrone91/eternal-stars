Dies ist die Abgabe-Version von Eternal-Stars. Feature-Liste
  • Einloggen mit erstellen Benutzeraccounts
  • Registrierung neuer Accounts
  • Systemrollen: Anonym, Noraml, Admin
  • Die Rechte der Rollen könne vom [SYS_ADMIN] (pw: admin) nach dem einloggen geändert werden
  • Nutzer können ihren Planeten verbessern und to die Produktion erhöhen
  • Nutzer können ihren Anzeigenamen, Email und Passwort unter Accoutn ändern
  • Über die Galaxie können die Nachbarn des Nutzers betrachtet werden
  • Nutzer können von Admin gelöscht werden
  • Die aktuellen Resourcen werden nur bei Bau eines Gebäudes gespeichert
